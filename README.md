# PayMe

A no-nonsense, fully configurable Rails credit card/payment vendor gem.

## Installation

Add this line to your application's Gemfile:

    gem 'pay_me'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pay_me
